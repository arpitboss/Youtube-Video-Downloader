# Youtube-Video-Downloader

This is a simple Python script that allows you to download YouTube videos. It uses the `pytube` library to handle the YouTube video download process.

# Prerequisites
Before using this script, make sure you have the following prerequisites installed on your system:

- Python 3 (https://www.python.org/downloads/)
- pytube library (can be installed using pip install pytube)

# Usage
- Enter the URL of the YouTube video you want to download when prompted.
- Select the desired video format and resolution from the available options.
- The script will start downloading the video and save it to the current directory.

# Notes
- The downloaded video will be saved in the same directory as the script.
- The available video formats and resolutions may vary depending on the YouTube video.
- Make sure you have the necessary permissions to download videos from YouTube.
- This script is intended for personal use only and should not be used to infringe on any copyrights or violate YouTube's terms of service.
